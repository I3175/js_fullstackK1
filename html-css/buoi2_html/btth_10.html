Đây là trang web của tôi

Chào mừng đến với trang web của tôi. Đây là một ví dụ về cách sử dụng thẻ img để hiển thị hình ảnh trên trang web.

Trên đây là mội hình ảnh ngẫu nhiên dược tải từ trang web Picsum. Bạn có thể thay đổi kích thước hình ảnh bằng cách chỉnh độ rộng và chiều cao trong thuộc tính 'width' và 'height' của thẻ ''. Bạn cũng có thể thêm một mô tả hình ảnh bằng thuộc tính 'alt'.
